## 三、分阶段实施计划

### Phase 1: 核心 AI 交互完善 (P0)

**目标**: 让 AI 真正"懂孩子"，实现 PRD 核心价值  
**预估**: ~2 周

#### 1.1 AI 年龄适配对话
- 后端 `ai.service.ts` 增加年龄感知 system prompt
- 3-4 岁: 简单短句 + 大量 emoji + 引导式提问
- 5-6 岁: 适当复杂度 + 知识拓展
- 前端传递用户年龄信息给后端

#### 1.2 AI 故事生成
- 新增 `POST /ai/story` 端点
- 结合兴趣标签 + 学科 + 年龄生成个性化故事
- 前端新增故事生成 UI

#### 1.3 内容安全过滤
- AI 回复增加儿童安全过滤中间件
- 敏感词检测 + 内容审核层
- 新增 `common/filters/content-filter.ts`

#### 1.4 个性化推荐展示
- 学生端首页展示推荐内容卡片
- 调用 `GET /recommend` API
- 新增 `RecommendedContent.tsx` 组件

#### 1.5 语音播放 UI
- 前端集成 `GET /voice/tts` API
- AI 回复增加"播放语音"按钮
- 新增 `VoicePlayer.tsx` 组件

### Phase 2: 学习体验闭环 (P0+P1)

**目标**: 完成"选内容 → 学习 → 评估 → 反馈"完整闭环  
**预估**: ~2 周

#### 2.1 互动练习引擎
- 新增选择题/拖拽题/连线题前端组件
- 基于内容 JSON 中的 interactive 字段渲染
- 新增 `components/quiz/` 目录

#### 2.2 成就系统可视化
- 新增 `AchievementShowcase.tsx` 展示页
- 徽章网格、星星计数、成长树
- 后端完善成就触发逻辑

#### 2.3 每日任务系统
- 后端基于推荐生成每日任务列表
- 前端展示任务进度和完成状态
- 新增 `DailyTask.tsx` 组件

#### 2.4 AI 学习评估反馈
- 完成学习后 AI 自动评估
- 给出个性化鼓励和建议
- 改进 `ContentDetail.tsx` 评估区域

#### 2.5 学习内容互动化
- JSON 内容中的交互元素前端渲染
- 新增 `ContentRenderer.tsx` 通用渲染器
- 支持 story/quiz/game/practice 类型

### Phase 3: 家长端完善 (P1)

**目标**: 家长端功能完整，数据可视化清晰  
**预估**: ~1.5 周

#### 3.1 五维能力雷达图
- recharts RadarChart 展示
- 语言/数学/科学/艺术/社会五维

#### 3.2 能力趋势曲线
- 各维度能力随时间变化折线图

#### 3.3 完整学习报告页
- 周报/月报详情页
- 总结 + 建议 + 对比参考

#### 3.4 家长控制完善
- 学习时段设置 UI
- 内容屏蔽管理
- 时长限制滑块

### Phase 4: 生产化部署 (P2)

**目标**: 从开发到可发布生产系统  
**预估**: ~1.5 周

#### 4.1 PostgreSQL 迁移
#### 4.2 Redis 缓存层
#### 4.3 Docker + docker-compose
#### 4.4 移动端响应式完善
#### 4.5 环境变量管理
