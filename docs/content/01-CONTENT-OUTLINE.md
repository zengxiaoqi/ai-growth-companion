## 内容格式模板

```yaml
# 示例：颜色宝宝
theme:
  id: lang_001
  name: 颜色宝宝
  age_range: 3-4
  domain: language
  duration: 5
  objectives:
    - 认识红、黄、蓝、绿四种基础颜色
    - 能说出颜色的名称
    - 能在生活中找到相应颜色的物品
  
  content:
    - type: story
      title: 彩色的世界
      text: |
        今天是彩虹王国的大日子！
        小红、小黄、小蓝和小绿要见面了...
      interactive:
        - type: question
          question: 红色像什么？
          options: [苹果, 天空, 草地]
          answer: 苹果
    
    - type: game
      title: 颜色配配对
      description: 把相同颜色的物品配对
      
    - type: practice
      title: 找颜色
      task: 在房间里找到红色的东西
```

## 内容安全标准

1. **语言**: 无暴力、无脏话、积极正向
2. **画面**: 色彩鲜艳但不刺眼、无恐怖元素
3. **价值观**: 友爱、勇敢、诚实、尊重
4. **年龄适配**: 严格按年龄推荐内容
